# MSc Project – Demand Forecast Uncertainty and Inventory Decisions

## Project Overview
This project investigates how demand forecast uncertainty affects inventory decisions in a retail environment. The study goes beyond traditional forecast accuracy by evaluating the operational impact of forecasting performance using inventory simulation.

## Objectives
- Prepare and analyse real retail transactional data
- Develop baseline statistical forecasting models
- Implement machine learning and deep learning approaches
- Generate probabilistic forecasts to capture uncertainty
- Evaluate operational performance using a base-stock inventory policy

## Repository Structure
- data/ – Sample or processed data (no large raw files)
- notebooks/ – Jupyter notebooks for analysis and modelling
- src/ – Python scripts and functions
- results/ – Model outputs and evaluation results
- figures/ – Visualisations used in the report
